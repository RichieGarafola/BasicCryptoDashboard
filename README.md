# Cryptocurrency Dashboard

This project showcases a simple cryptocurrency dashboard.

Cryptocurrency used in this platform:
- Bitcoin (BTC)
- Ethereum (ETH)
- Ripple (XRP)
- Bitcoin Cash (BCH)

---
